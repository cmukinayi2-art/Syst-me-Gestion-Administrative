# Système numérique de gestion administrative
Ce projet est un concept de plateforme numérique visant à améliorer la gestion administrative et le suivi des dossiers au sein des organisations et des PME.

## Fonctionnalités
- Gestion des utilisateurs
- Enregistrement des dossiers
- Suivi administratif
- Gestion des données
- Génération de rapports
  
## Technologies utilisées
- HTML
- CSS
- JavaScript
- PHP
- MySQL
  
## Objectif
L’objectif de ce système est d’aider les organisations à numériser leurs processus administratifs, à réduire le travail manuel et à améliorer l’efficacité dans la gestion des dossiers.
